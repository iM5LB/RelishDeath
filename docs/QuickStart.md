# Quick Start

## 1) Install

1. Install **PacketEvents** (required).
2. Put `RelishDeath-*.jar` in `plugins/`.
3. Restart the server.

Files created:
- `plugins/RelishDeath/config.yml`
- `plugins/RelishDeath/lang/en.yml`
- `plugins/RelishDeath/relishdeath.db` (SQLite)

## 2) Premium (Optional)

Buy a license key from the store and set it in `plugins/RelishDeath/config.yml`:

```yaml
license-key: "YOUR_KEY_HERE"
```

Then run:

```bash
/rd reload
/rd license
```

## 3) Basic Commands

```bash
/rd list
/rd locate
/rd restore
```

Premium:

```bash
/rd stash
/rd blackmarket
```
