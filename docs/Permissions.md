# Permissions

## Core

| Permission | Description | Default |
|---|---|---|
| `relishdeath.use` | Basic plugin usage | `true` |
| `relishdeath.restore` | Remote restore | `false` |
| `relishdeath.teleport` | Teleport to graves | `false` |

## Premium

| Permission | Description | Default |
|---|---|---|
| `relishdeath.trust` | Trust system commands | `true` |
| `relishdeath.marketplace.buy` | Buy from black market | `true` |

## Admin

| Permission | Description | Default |
|---|---|---|
| `relishdeath.admin.reload` | Reload configuration | `op` |
| `relishdeath.admin.backups` | Open backups GUI | `op` |
| `relishdeath.admin.license` | License status command | `op` |

## Timer Groups

Timer group permissions map to `timers.groups.*`:

| Permission | Group |
|---|---|
| `relishdeath.timer.vip` | `vip` |
| `relishdeath.timer.premium` | `premium` |
| `relishdeath.timer.admin` | `admin` |
| `relishdeath.timer.permanent` | `permanent` |
