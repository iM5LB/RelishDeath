# RelishDeath

Advanced death inventory protection plugin for Paper 1.21+.

## Key Features

### Free
- Active graves with real server-side entities (Geyser compatible)
- Holograms with countdown, items, XP, and death cause
- Particle guidance to graves (auto activates near your graves)
- Remote restore (permission-based)
- Void death protection (safe placement)
- Multi-world support
- Config validation and auto-recovery for bundled files

### Premium
- Stash (overflow items stored without overwriting inventory slots)
- Black market (expired graves can be purchased)
- Trust system (share grave access with trusted players)
- Admin death backups GUI (recovery tools)
- Economy integration (Vault, RelishCurrencies, SellGUI pricing)
